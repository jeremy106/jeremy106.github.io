Taghunter

Taghunter is an online scavenger hunt game where players locate street art in and around the city of Wellington, NZ.

This project uses the React framework to create a light-hearted, interactive UI with a backend powered by Node.js to manage the storage of game assets and user gameplay data.

This entire application was built from start to finish as part of a 7 day group project. While it still has a few rough edges, I'm proud of what my team achieved in such a short time. I intend to launch a mobile-friendly version of this app in the future
